# Offensive Python Journey

> Mi bitácora aprendiendo Python desde cero, con enfoque en hacking ofensivo, automatización, seguridad ofensiva y mentalidad Red Team.

![py-learning-journey](py_learning_journey.png)

---

### Sobre este repositorio

Este repo no es un curso copiado ni una colección de scripts sin contexto.

Es una bitácora donde documento mi proceso de aprendizaje en Python desde los fundamentos del lenguaje hasta su aplicación en automatización y seguridad ofensiva.

Aquí encontrarás tanto la parte teórica como la práctica, organizadas para seguir una ruta de aprendizaje progresiva.

---

### Objetivo

Construir una base sólida en Python aplicada a:

* scripting ofensivo
* automatización
* networking
* manejo de archivos
* parsing
* concurrencia
* desarrollo de herramientas
* pentesting
* offensive security

La idea no es solo aprender Python.

La idea es desarrollar la forma de pensar necesaria para construir herramientas.
